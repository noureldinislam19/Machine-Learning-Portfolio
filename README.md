- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Regression
- Classification
- Clustering
- Model Evaluation
- Neural Networks
- Ensemble Learning
- Dimensionality Reduction

## Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

Each project contains:
- Problem Statement
- Dataset Description

- Model Implementation
- Evaluation Metrics
- Results & Discussion
- Future Improvements

## Goals

- Build a strong foundation in Machine Learning.
